# Go ToDo List

Go ToDo List — это простое CLI-приложение на чистом Go для управления списком задач. Позволяет создавать, просматривать, редактировать и удалять задачи, сохраняя их в файл.

## Возможности
- Создание новой задачи
- Просмотр всех задач
- Редактирование задачи
- Удаление задачи
- Сохранение и загрузка задач из файла

## Технологии
- **Язык**: Go
- **Хранение**: Локальный текстовый файл

## Установка
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/aliskhannn/go-todo-list.git
   cd go-todo-list
   ```

2. Запустите приложение:
   ```bash
   go run .
   ```

## Использование
После запуска выберите опцию в меню:

```
Choose an option:

v - view tasks
c - create task
e - edit task
d - delete task
q - quit
```

- **v**: Показать все задачи
- **c**: Добавить новую задачу
- **e**: Изменить существующую задачу
- **d**: Удалить задачу
- **q**: Выйти из приложения

## Структура
Все файлы находятся в корневой директории:
- `main.go` — основной код приложения
- `tasks.go` — методы для управления задачами
- `tasks.txt` — файл для хранения задач
